

Все исходники лежат в корне сайта (папка «src»)

Все компоненты находятся в папке «components»

Для внесения изменений в компонент необходимо развернуть у себя локальную версию:

1. установка npm-пакетов: npm install. 

2. Запуск dev-версии: npm run dev

После внесения изменений в компоненты необходимо запустить сборку всех файлов проекта с помощью команды npm run build:prod

Минифицированные файлы css и js будут располагаться в папке «build», которыми нужно будет заменить старые версии файлов на сайте.
